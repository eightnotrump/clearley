clearley.lib documentation

More fns and macros for context-free grammars.

angle-brackets

(angle-brackets a-rule)(angle-brackets a-rule action)(angle-brackets name a-rule action)
Surround a rule in angle brackets: <>.

braces

(braces a-rule)(braces a-rule action)(braces name a-rule action)
Surround a rule in braces: {}.

brackets

(brackets a-rule)(brackets a-rule action)(brackets name a-rule action)
Surround a rule in brackets: [].

canonical-natnum

Matches a positive number with no leading zeroes,
returning the number. (0 is a matching number.)

char-range

(char-range min max)(char-range min max action)
Creates a rule that accepts any one character within a given range
given by min and max, inclusive. min and max should be chars. The default
action is the identity.

char-to-num

(char-to-num char)(char-to-num char char-start num-start)
Maps chars to numbers linearally. Default maps \0 to 0.
This behavior can be overriden by providing char-start and num-start,
such that char-start -> numstart (for example \a -> 10).

digit

A rule that matches an input digit, and returns a number for it.

digit1-9

Like digit but matches 1 through 9.

hex-digit

A rule that matches an input hex digit, case insensitive, and returns
a number for it.

make-num

(make-num digits)(make-num digits radix)
Turns a bunch of digits into a number.
An optional radix can be supplied (default 10).
Useful for plugging into parse actions.

natnum

Matches a positive number, returning the number. Any number of leading zeroes
is allowed.

opt

(opt subrule)(opt subrule action)(opt name subrule action)(opt name subrule action default)
Creates a rule that matches a subrule, or nothing. The action will be passed
the subrule's value, or a default value (nil if unspecified).
The default action is the identity.

or-rule

(or-rule rules)(or-rule rules action)(or-rule name rules action)
Creates a rule that matches one of some number of given rules. The default
action is the identity.

parens

(parens a-rule)(parens a-rule action)(parens name a-rule action)
Surround a rule in parens: ().

plus

(plus subrule)(plus subrule action)
Creates a rule that matches a subrule one or more times.
The defalt action returns a seq of the matches

quotes

(quotes a-rule)(quotes a-rule action)(quotes name a-rule action)
Surround a rule in quotes.

single-quotes

(single-quotes a-rule)(single-quotes a-rule action)(single-quotes name a-rule action)
Surround a rule in single quotes.

star

(star subrule)(star subrule action)(star name subrule action)
Creates a rule that matches a subrule zero or more times.
The default action returns a seq of the matches.

string-rule

(string-rule str)(string-rule str action)
Creates a rule that matches some string. The default action returns the string.

surround

(surround r a-rule)(surround r1 a-rule r2)(surround r1 a-rule r2 action)(surround name r1 a-rule r2 action)
Surrounds a rule with one or two other rules.
Returns the result of the center rule.
Useful for warpping in quotes, braces, brackets, &c.

symbol-rule

(symbol-rule a-symbol)(symbol-rule a-symbol action)(symbol-rule name a-symbol action)
Creates a rule that points to some other rule, identified by the given symbol.
The default action is the identity.

token

(token a-token)(token a-token action)(token name a-token action)
Creates a rule that matches a token. The default action returns the token.

zero

Matches \0 and returns 0.
